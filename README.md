# Sweep the Floor Cleaning Co. Website

Static HTML/CSS/JS website draft based on the provided Figma Make screenshots.

## Files
- `index.html` — homepage
- `services.html` — services page
- `about.html` — about page
- `contact.html` — contact / quote request page
- `styles.css` — all styling
- `script.js` — FAQ accordion, mobile nav-ready interactions, and scroll/animation behavior
- `assets/` — logo and image assets
- `robots.txt`, `sitemap.xml` — basic SEO files

## How to open in VS Code
1. Unzip this folder.
2. Open VS Code.
3. Go to **File > Open Folder**.
4. Select the unzipped `stf_rebuild` folder.
5. Install/use the **Live Server** extension, then right-click `index.html` and choose **Open with Live Server**.

## Notes
- Several images are loaded from Unsplash URLs as placeholders. Replace them with final client photos before deployment.
- The site is static and can be deployed directly to Netlify.
